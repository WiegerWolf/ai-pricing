About This Comparison

This table compares various AI language models across different metrics:

  • Arena ELO: A comparative score measuring model performance (higher is better)
  • Coding ELO: Specific performance score for coding tasks (higher is better)
  • Speed: Tokens processed per second
  • Context: Maximum input size in thousands of tokens
  • Pricing: Cost per million tokens for input and output separately
